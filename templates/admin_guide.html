Admin Guide

Welcome to the FaceCheck Admin Panel! Below is a quick guide for each feature available on your dashboard:

  1. Mark Attendance: Capture student attendance in real-time using the camera. Ensure students are registered and their faces are visible for accurate recognition.
  2. Student Database: View all students, filter by roll number or class. You can also delete all student records.
  3. Attendance Records: View historical attendance data. Filter by date or student roll number to monitor attendance trends.
  4. Set Total Lectures: Update the total number of lectures for each subjects. This will reflect in attendance summaries and reports.
  5. Attendance Summary: Make sure to upadte total lectures for each subject in "Set Total Lectures" to, Review summarized attendance for each student and class. Check attendance percentages and trends.
  6. Admin Settings: Change your password or add additional admin accounts. Manage existing admin accounts safely.
Important: At the beginning of each new academic year, ensure that the Subjects table in "Mark Attendance", the Attendance table in "Attendance Records", and the Students table in "Student Database" are cleared. This avoids confusion between old and new records. After clearing, all students will need to re-register to participate in attendance marking.
